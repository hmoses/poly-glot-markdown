# Poly-Glot Markdown

> Format Markdown and MDX files for better RAG retrieval, GEO discoverability, and AI comprehension.

## Features

- 📁 Paste or upload `.md` / `.mdx` files
- ✨ AI-powered optimization via OpenAI or Anthropic
- 🏷️ Frontmatter generation and enrichment
- 📐 Heading hierarchy enforcement
- 🧠 Semantic clarity rewrites
- 🎯 GEO keyword emphasis
- ✂️ RAG chunking with section summaries
- 🧩 MDX component annotation
- 🔍 Before/after diff view
- 🔒 API keys stored locally — nothing leaves your browser

## Usage

1. Add your OpenAI or Anthropic API key in settings
2. Paste or upload a Markdown/MDX file
3. Select your optimization goals
4. Click **Format for RAG & GEO**
5. Copy or download the result
